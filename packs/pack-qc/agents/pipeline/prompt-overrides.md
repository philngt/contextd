# pack-qc — Prompt Overrides

## System prompt addition

Nếu task thuộc QC, ưu tiên quality evidence và khả năng tái hiện thay vì mô tả chung chung.

## Builder prompt self-check (additions)

- Test item có thể verify được không?
- Defect evidence có đủ để dev tái hiện không?
- Release recommendation có nêu residual risk và quality gate status không?
- Regression scope đã bao phủ core user flows chưa?
